# Grid Visualizer + Whiteboard

An interactive grid editor with a whiteboard overlay.
Built using React + Vite.

## Features
- Dynamic grid rendering from arrays
- Editable grid cells
- Whiteboard pen & eraser
- Adjustable pen/eraser size
- Scrollable multi-grid layout
  
##Tech Stack :
Frontend: React
Build Tool: Vite
Styling: Inline CSS
Canvas API: For whiteboard drawing
Version Control: Git + GitHub

## Run locally
npm install
npm run dev

## Build
npm run build
